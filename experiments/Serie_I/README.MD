# Série I — Inteligência em Regimes Relacionais

Esta série investiga como sistemas inteligentes operam sob divergência
observacional, sem acesso a um estado global verdadeiro.

Os experimentos mostram que:
- Divergência não implica falha operacional.
- Alinhamento rígido ignora incerteza.
- Concessão probabilística reduz risco ao custo de margem.
- Inteligência é governança de regimes, não descoberta de verdade.

A Série I conecta o Framework CMR a:
- IA e alinhamento
- Gestão de tráfego aéreo
- Sistemas multi-sensores
- Tomada de decisão sob incerteza
